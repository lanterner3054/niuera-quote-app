# NIUERA 报价工作台更新说明

## v3 - 2026-07-02

### 这次加了什么

- 新增「自定义产品」入口:非标准品不需要先写入产品库,可直接加入报价明细。
- 报价明细里的产品信息可临时编辑:产品名称、描述、单位、图片文件名、数量、单价、分档价格都可以在当前报价单内修改。
- 删除客户公司和联系人输入框里的示例客户内容,新建报价时客户信息保持空白。
- 生成报价前拦截空价产品,避免导出单价为空的报价单。
- 后端同步校验产品名称和价格,即使绕过前端也不会生成明显缺字段的报价。
- 修复 PI 编号不同步风险:当前 `next_seq` 已校正为 `260071`,生成时由服务端按历史输出文件和配置共同取下一个可用编号。
- PI 号生成改为服务端锁内占号,降低局域网多人同时生成时撞号的风险。
- `requirements.txt` 补齐图片上传/处理依赖:
  - `python-multipart>=0.0.9`
  - `Pillow>=10.0`
- 乱码文件名 `鏇存柊璇存槑_v2.txt` 已更正为 `更新说明_v2.txt`。

### 使用提醒

- 当前形态仍是局域网常驻服务:本机运行 `start.bat`,同事访问 `http://<内网IP>:8000`。
- `start.bat` 中的 `<内网IP>` 是固定 IP,不要改回自动检测。
- 更新代码后需要重启一次 `start.bat` 才会生效。
- 若迁移到新电脑,先安装依赖:

```bat
pip install -r requirements.txt
```

## v2 - 新增产品图片

### 这次加了什么

- 产品管理界面每行多了「图片」列:点「上传」选图即绑定,显示缩略图,可清除。
- 生成报价时,产品图自动插进 Photo 列并缩放;没图就留空。
- 任何 Excel/WPS 都能正常显示,不用云端函数。

### 用法

- 右上「产品管理」-> 找到产品 -> 该行「上传」选图 ->「保存到 products.json」。
- 之后报价里选到这个产品,图就自动出现在 Photo 列。
- 图片会存在 `data/images/` 里。
